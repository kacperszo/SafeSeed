# SafeSeed

Safe seed is a mental-health centered app that was built during HackYeah hackathon by the prisme team.

Features:

-   Live Chat
-   User registration

Technologies:

-   Next.js
-   React
-   Tailwind.CSS
-   Formik
-   React Query
-   Socket.io
